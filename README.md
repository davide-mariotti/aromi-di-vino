# Aromi di Vino

**Aromi di Vino** is a modern, responsive, and feature-rich e-commerce website for a wine shop. It is built with **HTML5**, **CSS3**, and **JavaScript**, following best practices for web development.

## 🚀 Features

- **Modern Design**: Clean, professional, and visually appealing interface.
- **Responsive Layout**: Adapts seamlessly to all devices (desktop, tablet, mobile).
- **Interactive Elements**: Smooth animations, hover effects, and interactive components.
- **Product Showcase**: Dedicated pages for wine products with detailed information.
- **Shopping Cart**: Functional cart system to add, remove, and manage products.
- **User Authentication**: Secure login and registration pages.
- **Contact Form**: Easy-to-use contact form with validation.
- **Navigation**: Intuitive navigation with dropdown menus and clear calls to action.
- **Accessibility**: Built with accessibility standards in mind.

## 🛠️ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).
- A text editor or IDE (VS Code, Sublime Text, etc.).

### Installation

1.  **Clone the repository** (or download the source code).
2.  **Open the project** in your web browser by navigating to `index.html`.

## 🚀 Usage

### Homepage (`index.html`)

The homepage serves as the entry point to the website. It features:
- Hero section with a call to action.
- Featured products section.
- Customer testimonials.
- Newsletter signup.

### Products Page (`products.html`)

Browse all available wines with detailed information:
- Product name and description.
- Price and availability.
- Add to cart functionality.

### Shopping Cart (`cart.html`)

Manage your selected products before checkout:
- View items in cart.
- Update quantities.
- Remove items.
- Proceed to checkout.

### Checkout (`checkout.html`)

Complete your purchase:
- Shipping information.
- Payment details.
- Order summary.

## 🎨 Styling

The website uses a modern and professional color palette:
- **Primary Color**: `#8B0000` (Dark Red) - Evokes wine and luxury.
- **Secondary Color**: `#FFD700` (Gold) - For accents and highlights.
- **Background**: `#F8F8F8` (Light Gray) - Clean and modern.
- **Text**: `#333333` (Dark Gray) - High readability.

The design is fully responsive and tested on various screen sizes.

## 📱 Responsive Design

The website uses a mobile-first approach with media queries to ensure optimal viewing on all devices:

- **Mobile (< 600px)**: Single-column layout, touch-friendly navigation.
- **Tablet (600px - 900px)**: Two-column layouts, optimized spacing.
- **Desktop (≥ 900px)**: Full multi-column layouts with enhanced visuals.
